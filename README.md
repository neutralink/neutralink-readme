# 🌍 NeutraLink — Project Index & Ecosystem Overview

Welcome to the official repository index for **NeutraLink**, a Web3 platform that connects **solar energy**, **IoT devices**, and **blockchain** to democratize access to **carbon credits**.

> Empowering people and businesses to generate, trade, and certify real environmental impact.

---

## 💡 What is NeutraLink?

**NeutraLink** transforms real-time solar generation into **tokenized carbon credits (NTL)** through a secure and transparent process.  
From IoT monitoring to blockchain certification, we provide the infrastructure for accessible and verifiable environmental compensation.

## 🧩 Technologies

- **Solidity** (ERC-20, UUPS Proxy, Role Control)
- **React / Next.js** (Web interface)
- **React Native / Expo** (Mobile)
- **ESP32 / RS485 / HMAC-SHA256** (IoT)
- **Node.js / Express** (Backend API)
- **Web3.js / Ethers.js** (Blockchain Integration)
- **PostgreSQL / MongoDB** (Databases)

---

## 🌎 Use Cases

- 🧑‍🌾 **Solar Generators** monetize verified production  
- 🏢 **Companies** offset emissions via certified credits  
- 🛠️ **Installers & Integrators** offer the full solution to clients  
- 🧾 **Certifiers** ensure traceable impact  
- 🛒 **Buyers** trade or retire NTL with transparency

---

## 🔐 Security

- Role-based access (Generator, Certifier, Auditor, Admin)  
- All contracts upgradeable via **UUPS Proxy**  
- IoT data signed using **HMAC-SHA256**  
- Credits tied to real data, not speculation
